# 资源文件夹

此文件夹用于存放应用的资源文件。

## 需要的资源

### 图标文件

为了正确打包应用，你需要准备以下图标：

1. **icon.ico** (Windows 图标)
   - 尺寸: 256x256
   - 格式: ICO
   - 用途: Windows 应用图标

2. **icon.png** (通用图标)
   - 尺寸: 512x512
   - 格式: PNG
   - 用途: 开发时使用，显示在任务栏

## 如何生成图标

### 在线工具

- [ICO Converter](https://icoconvert.com/) - 将 PNG 转换为 ICO
- [Favicon Generator](https://realfavicongenerator.net/) - 生成多种尺寸图标

### 使用 Photoshop/GIMP

1. 创建 512x512 的画布
2. 设计你的图标
3. 导出为 PNG 格式
4. 使用在线工具转换为 ICO

## 临时方案

如果暂时没有图标，应用仍然可以正常运行和打包，只是会使用默认图标。

## 推荐设计

建议使用与"面试助手"主题相关的图标元素：
- 🔥 火焰（热门、热点）
- 💡 灯泡（智能、想法）
- 🎯 靶心（精准、目标）
- 🤖 机器人（AI）
- 📸 相机（截图）

可以组合这些元素创建独特的图标。

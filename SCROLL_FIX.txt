🔍 问题

滚动到最下面时，所有内容消失，变成黑框。

根本原因：
→ .overlay 设置了 overflow: hidden
→ 导致滚动时内容被裁剪掉
→ 只能看到窗口范围内的内容
→ 滚动过头就看到空白/黑色

═══════════════════════════════════════════════════════════════════════════

🔧 修复方案

1. 移除 overflow: hidden
   之前：
   ```css
   .overlay {
     overflow: hidden;  ← 问题！
   }
   ```
   
   现在：
   ```css
   .overlay {
     /* 不设置 overflow: hidden */
   }
   ```

2. 确保滚动容器正确
   ```css
   .overlay-content-wrapper {
     flex: 1;
     overflow-y: auto;  ← 滚动在这里
     overflow-x: hidden;
   }
   ```

3. 添加可见的滚动条
   → 8px 宽度
   → 紫色滚动条
   → 悬停高亮

欢迎使用 AI 面试助手！这是一个强大的 Windows 桌面应用。

═══════════════════════════════════════════════════════════════════════════

📦 第一步：安装依赖

    双击运行： install.bat

    这将自动：
    ✓ 安装前端依赖（Node.js）
    ✓ 创建 Python 虚拟环境
    ✓ 安装后端依赖（Python）
    ✓ 创建配置文件

═══════════════════════════════════════════════════════════════════════════

🔑 第二步：配置 API Key

    1. 打开文件： backend\.env
    2. 找到这行： OPENAI_API_KEY=your_openai_api_key_here
    3. 替换为你的真实 API Key
    4. 保存文件

    💡 如何获取 API Key？
    访问：https://platform.openai.com/api-keys

═══════════════════════════════════════════════════════════════════════════

🚀 第三步：启动应用

    双击运行： start-all.bat

    这将启动：
    ✓ 后端服务（FastAPI - 端口 8000）
    ✓ 前端应用（Electron + React）
    ✓ 主窗口（显示详细信息）
    ✓ 悬浮窗（右上角，永远置顶）

═══════════════════════════════════════════════════════════════════════════

⌨️ 快捷键

    Ctrl + H       → 全屏截图
    Ctrl + Enter   → 发送截图给 AI 分析

═══════════════════════════════════════════════════════════════════════════

🎯 使用流程

    1. 按 Ctrl+H 截图
       → 截图显示在悬浮窗中

    2. 按 Ctrl+Enter 分析
       → AI 开始分析截图内容

    3. 查看结果
       → 悬浮窗显示简短回复
       → 主窗口显示完整回复

❓ 遇到问题？

    问题：快捷键不工作
    → 重启应用，检查是否有其他程序占用快捷键

    问题：后端连接失败
    → 确保后端服务在运行，访问 http://127.0.0.1:8000/health

    问题：API 调用失败
    → 检查 backend\.env 中的 API Key 是否正确

    问题：缺少依赖
    → 重新运行 install.bat

═══════════════════════════════════════════════════════════════════════════

💡 使用提示

    1. 第一次启动可能需要几分钟下载依赖
    2. 确保网络连接稳定（需要访问 OpenAI API）
    3. 悬浮窗可以点击顶部按钮最小化/展开
    4. 建议在运行应用前先配置好 API Key

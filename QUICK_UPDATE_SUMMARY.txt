✅ 已完成的所有优化

1. 📍 位置优化
   ✓ 从屏幕居中 → 改为置顶在最上方
   ✓ 水平居中，垂直 y=0
   ✓ 不遮挡屏幕中间的重要内容

2. 📏 尺寸优化
   ✓ 宽度：屏幕宽度的 50%
   ✓ 初始高度：80px（只显示按钮）
   ✓ 最大高度：屏幕高度的 50%
   ✓ 根据内容自动调整

3. 🌫️ 透明度
   ✓ 默认 50% 透明
   ✓ 可以透过窗口看到背后内容
   ✓ 不会完全遮挡

4. 🎨 显示逻辑
   ✓ 默认只显示：截图、分析、隐藏 三个按钮
   ✓ 截图后自动展开显示截图
   ✓ 收到回复后继续展开显示 AI 回复
   ✓ 内容自适应滚动

5. ⌨️ 快捷键
   ✓ Ctrl+H：截图
   ✓ Ctrl+Enter：分析
   ✓ Ctrl+B：隐藏/显示悬浮窗

🚀 如何使用

步骤 1：重启应用
→ 关闭所有窗口
→ 运行 start-all.bat

步骤 2：测试功能
→ 看到顶部只有三个按钮（截图、分析、隐藏）
→ 按 Ctrl+H 截图，窗口自动展开显示截图
→ 按 Ctrl+Enter 分析，窗口继续扩展显示 AI 回复
→ 按 Ctrl+B 隐藏窗口

🔍 立即检查步骤

步骤 1：完全重启应用
→ 关闭所有 Electron 窗口
→ 停止后端（Ctrl+C）
→ 运行 start-all.bat

步骤 2：检查窗口是否存在
→ 按 Alt+Tab 查看所有窗口
→ 应该能看到 "AI Interview Assistant" 窗口
→ 窗口应该在屏幕顶部中间

步骤 3：打开 DevTools
→ 在窗口上按 F12
→ 切换到 Console 标签
→ 查看是否有错误

步骤 4：查看日志
→ 应该看到 "Overlay 渲染状态:" 日志
→ 复制完整日志给我

🔧 可能的问题

问题 1：窗口太小看不到
→ 初始高度 80px
→ 可能被任务栏遮挡
→ 或者在屏幕外

问题 2：透明度太高
→ 设置了 50% 透明
→ 可能看不清楚

问题 3：CSS 加载失败
→ 样式没有应用
→ 内容可能是白色/透明

问题 4：React 渲染失败
→ JavaScript 错误
→ 组件没有挂载

🚀 临时解决方案

如果找不到窗口，尝试：

1. 查看所有窗口
   → 按 Alt+Tab
   → 找 "AI Interview Assistant"

2. 查看任务栏
   → 右键 Electron 图标
   → 选择 "Overlay" 窗口

3. 查看进程
   → 打开任务管理器
   → 查找 Electron 进程

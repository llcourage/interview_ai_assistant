# 更新日志

## [1.0.0] - 2024-11-27

### ✨ 新增功能

- 🎯 全局快捷键支持（Ctrl+H 截图，Ctrl+Enter 分析）
- 🎨 悬浮窗界面（永远置顶、半透明、可折叠）
- 🤖 OpenAI Vision API 集成
- 📸 全屏截图功能
- 💬 AI 智能分析（支持面试题、算法题、系统设计等）
- 🖥️ 主界面（显示详细信息和完整回复）
- 🔧 完整的后端 API（FastAPI）
- 📦 一键安装和启动脚本
- 📚 详细的使用文档

### 🏗️ 技术实现

- Electron + React 前端
- FastAPI Python 后端
- TypeScript 类型支持
- Vite 构建工具
- IPC 进程间通信
- CORS 跨域支持

### 📄 文档

- README.md - 项目说明
- USAGE.md - 使用指南
- backend/README.md - 后端文档
- 完整的代码注释

### 🎁 开箱即用

- install.bat - 一键安装依赖
- start-all.bat - 一键启动应用
- build.bat - 一键打包发布
